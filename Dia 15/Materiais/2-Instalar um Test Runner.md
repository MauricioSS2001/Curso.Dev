# Instalar um Test Runner

Se você ainda não passou pela **jornada dos Testes Automatizados**, com o que eu vou te mostrar daqui para frente, provavelmente você vai pensar _"nossa... testes automatizados é só isso!?!"_ ... aí naturalmente você vai enfiando cada vez mais o pé na lama, talvez num ponto de começar a colocar o assunto Testes Automatizados num **pedestal**. Aí depois quando dá aquele _click_ de novo sobre o que realmente está acontecendo... você vai voltar a pensar: _"nossa… testes automatizados… é só isso?!??!?"_

É uma jornada **muito legal** e reveladora de vários aspectos sobre a "arte de programar", e tudo começa com a instalação de um **Test Runner** 💪
